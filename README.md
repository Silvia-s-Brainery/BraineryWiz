## Instalation Stpes:
- Using command prompt using the following command:

```
pip install BraineryWiz
```
- For anaconda users:

```
conda ...
```

## How to use:
- User should import package: 

```
import BraineryWiz as bz
```
- for Jupyter users:

```
import BraineryWiz.Jupyter as bz
```
